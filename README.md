# A full-stack Real Estate web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).
# This platform allows users to browse, search, and manage property listings for buying, selling, or renting homes.

# The project demonstrates modern full-stack development with REST APIs, authentication, database management, and responsive UI design. Users can create accounts, upload property listings, and explore available properties with detailed information and images. Real estate platforms typically provide listing management, search filters, and user accounts to connect buyers with sellers.

# Run backend:
  cd server
  npm run dev

# Run frontend:
  cd client
  npm run dev

# Run Socket:
  cd socket
  npm run dev

# Open in browser:
  http://localhost:3000
